# Spring Boot

Dependencias Proyecto / Starters
* Oracle Driver
* Spring Data JPA
* Spring Web
* Spring Boot Dev Tools

Aplicación API REST con acceso de datos Oracle XE para persistir la información

El acceso se puede realizar desde Postman o Navegador.
